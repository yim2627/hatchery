# 접근성 규칙 — Web

- 시맨틱 HTML 요소(`button`, `nav`, `main`, `section` 등)를 사용한다.
- 커스텀 인터랙티브 요소에 `aria-label` 또는 `aria-labelledby`를 제공한다.
- 모든 인터랙티브 요소에 키보드 네비게이션이 작동하는지 확인한다.
- 포커스를 관리한다: 모달/다이얼로그 열 때 포커스 이동, 닫을 때 복원.
- 네이티브 시맨틱 요소가 맞지 않을 때만 `role` 속성을 사용한다.
- 스크린 리더(VoiceOver, NVDA)와 키보드 전용 네비게이션으로 테스트한다.
